                                Проверки ПЧ

Через анкету:

1.  Проверка поля для ввода ПЧ (с корректными и некорректными данными)
2.  Проверка переходов на другой вид оформления сделки после ввода ПЧ
3.  Проверка полей для ввода характеристик товара после ввода ПЧ
4.  Проверка удаления одного из товаров ПЧ (при удалении должны удалиться все товары)
5.  Проверка работы ПЧ со смежными сделками
6.  Проверка изменения количества товаров
7.  Проверка ввода нескольких ПЧ
8.  Проверка ввода дубликатов
9.  При изменении характеристики товара проверить проверку при выборе схемы кредита
10. Проверка отображения корректности заполненных данных при печати
11. Проверка корректности заполненных данных в базе данных



Через договор комиссии:
 
1.  Проверка поля для ввода ПЧ (с корректными и некорректными данными)
2.  Проверка возможности вводить только ПЧ с услугами
3.  Проверка полей для ввода характеристик товара после ввода ПЧ
4.  Проверка удаления одного из товаров ПЧ (при удалении должны удалиться все товары)
5.  Проверка изменения количества услуг
6.  Проверка ввода нескольких ПЧ
7.  Проверка ввода дубликатов
8.  При изменении характеристики товара проверить проверку при выборе схемы кредита
9.  Проверка отображения корректности заполненных данных при печати
10. Проверка корректности заполненных данных в базе данных
